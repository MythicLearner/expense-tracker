# Expense Tracker

A beginner-friendly Python project that records expenses in an SQLite database.

## How to Run
1. Install Python 3.10 or higher
2. Run:
   ```bash
   python main.py
3. follow the prompts to add, view, edit, delete expenses, and view totals by preset categories amongst other things
4. Table Schema
Column    | 	Type    |	Description
id	      |  INTEGER  |  primary key
date	    |  TEXT	    |  timestamp
amount	  |  REAL	    |  expense amount
category	|  TEXT	    |  expense category
note	    |  TEXT	    |  optional note
5. Add JavaScript/Electron interface for desktop visualization.
